Новый бложик стартует здесь

November 25, 2011

Наконец удалось запилить бложик. С использованием механизма хостинга Github для статичных страниц. Сами страницы генерируются с помощью гем-сервера jekyll, а посты пишутся на языке разметки markdown.

Надо сказать, что идею подобного обустройства персонального блога я спёр у CTO компании, в которой нынче работаю. Сам он позаимствовал её у автора jekyll. А различные плагины для jekyll, кстати говоря, можно брать оттуда же.

Плагин для подгрузи фотоальбомов заданного юзера из flickr я решил заменить на аналог, работающий с picasa. Поскольку готового варианта для него не нашлось, а плагин, который предлагает сам гугл, приспособить никак не удалось (просто потому, что я не знаю, как прицепиь плагин к jekyll-приложению), пришлось соорудить гем, и взять на себя почётную роль автора. Теперь он доступен каждому, а называет просто: google-picasa.

Рассказывать про то, как устроен и работает сам jekyll, я не буду. Хотя работает он здорово. Просто про это уже хорошо написано тут и тут. Так что всем удачи.

Сайт создан на базе движка для статичных страниц и хостится на github static pages. Домен принадлежит мне с 2008 года.
